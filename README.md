##世界观

这是一个名叫“互联世界”的世界中，国家的运作与发展采用的“神明制”。在神界，神明并不是永生的，如果信奉人少，香火不旺，则神明会灵力大损，逐渐变成落魄神明；如果被芸芸众生逐渐遗忘，神明就会消亡。反之，如同佛祖，耶稣这样的大神明，正因有宏伟的神殿，绵远不绝的信奉和香火，所以在能位于众神之上。

这个世界也是类似的运作机制，国家如神明，国家的发展与运作离不开一个名叫“泉钻”的晶体，此晶体具有巨大的能量，是维持国民生命值以及锻造“品器”必不可少的物质，一个国家的强大与否往往就看“泉钻”的存量有多少。而“泉钻”的成型必须要有大量的名为“泉”的灵力（人们称之为“泉之力”）凝练而成，而“泉之力”只有白精灵才能释放。但是，精灵不会平白无故释放“泉之力”，需要通过“品器”刺激，因为人们意外发现精灵在产生各类情感共振的时候会不自主释放“泉之力”，例如，“腾之国”的“品器”可以让精灵找到迅速联系到其他同伴产生愉快之情而释放“泉之力”，然后被“品器”吸收。

“品器”共分9阶，9品最强。一般三大国都有至少1个9阶“品器”。好的“品器”可以源源不断吸收多达亿万“泉之力”，并且稳定异常，不会发生大量“泉之力”逃逸的情况。一个高品阶的“品器”足以让一个无名小国一跃成为独角兽之国。

国家的强盛与否与“泉之力”的多少并无直接关系，关键要看是否要强大的把“泉之力”凝练成“泉钻”晶体的能力。有一些国家，虽然“泉之力”不算很多，但凝练能力异常强大，其实力也是相当强悍，如一些名为“游之国”的国家；还有的国家通过透支“泉钻”吸收了很多“泉之力”，但说不定只是强弩之末，例如以前闻名一时的“团之国”。

因此，国家要想屹立于冬至大陆，一定要想办法锻造高品阶的“品器”。什么是高品阶的“品器”呢？一是外表对精灵很有吸引力，例如，幼齿白精灵很喜欢圆形的物体，圆形的对其就很有吸引力，二是，“品器”能够非常简单便捷地让精灵产生“情感共振”，或刺激的或喜悦的，因为精灵总是游荡不定的，如果情感波动产生很费时费力，精灵早就漂到其他地方或者直接消失了；三是对“泉之力”的吸收与保存要稳定，“泉之力”具有灵性，总是想办法逃窜，如果“品器”有漏洞，或者突然崩溃，“泉之力”就会迅速大量流失，等于没有用；四是自身要坚固，因为周围总会有其他“品器”，激烈的碰撞和争斗在所难免，如果像妙脆角那样嘎吱脆，再多的“泉钻”都是白搭。

因此，锻造“品器”是一件非常不容易的事情，需要大量的时间，也需要消耗大量珍贵的“泉钻”，更关键的是，需要众多具有对“品器”锻造有特殊能力的人力。

##故事背景
“阅之国”乃东之大陆冉冉升起的新兴国家，分很多学院，“哇牛叉”学院为其中之一，“哇牛叉”学院又分“沪江总院”和“京城分院”，每个学院又分两个社团，前端公社和设计公社。“哇牛叉”学院的主要职责就是锻造“阅之国”几乎所有“品器”的外部构造，尽可能吸引精灵，尽可能以简单的方式释放“精神共振”，并且快速吸引“泉之力”。

而本故事讲述的就是“哇牛叉”学院中那些对“品器”的外部锻造具有超能力的伙伴们的故事。
