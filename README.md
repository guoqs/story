世界观和故事背景
---------------

这是一个精灵与人共存的世界。


人们把精灵大规模出现的那一年称为“互联元年”，至今已经过去22年了。


精灵出现的意义上让这个世界出现了一个全新的能量循环体系，而精灵所扮演的角色就是整个循环体系重要的媒介和信使。

风雨土木这些自然之物随着日月滋养会释放“自然灵气”，在过去，“自然灵气”会慢慢稀释在大气中，最终逃逸到地球之外。但精灵出现改变了这一现状，因为可以吸收“自然灵气”，并转换成名为“泉”的“灵力”，人们称之为“泉之力”。

“泉之力”进入人体之后，可以使人精力充沛，精神焕发，更有一些人会具有异能。

绝大多数人，“泉之力”只能通过口鼻进入，但人的呼吸是有限的，这种吸收方式效率非常低。

但是，“泉之力”可以通过特殊方式结晶，这种结晶体称为“泉钻”，具有大量的精纯的“灵力”能量，作用非常广泛。已经成为这个世界通用的最高级的流通货币。

因此，当下，各个国家所有的合作或者冲突根源都离不开“泉钻”。

如果“泉之力”足够，人是可以运功凝结出“泉钻”的，但只能自用。站在国家的角度，必须有更加高效的东西进行生产、管理和运作。这个东西就是“品器”。

前面讲过了，人这个个体吸收“泉之力”和凝结“泉钻”效率很低，但是“品器”则完全不可同日而语，从“品器”这两个汉字就可以看出其“口”之多，吸收能力之强，并且可以很短时间凝结成“泉钻”。

“品器”共分9阶，9品最强。好的“品器”可以源源不断吸收多达亿万“泉之力”，并直接凝结成“泉钻”，并且稳定异常，不会发生大量“泉之力”逃逸的情况。目前“互联直接”几个大国都有至少1个9阶“品器”，而一个高品阶的“品器”足以让一个无名小国一跃成为独角兽之国。

可以看出，一个国家能否繁荣强盛，“品器”至关重要。

然而，好的“品器”的锻造非一朝一夕一个人能够完成，因此，国家会设立机构专门设计、生产与使用“品器”，而给这些设计与生产“品器”的报酬就是“泉钻”。

由于“品器”必须在外使用，难免会遇到劫匪或者他国敌人或者“黑精灵”，因此，都只招收可以利用“泉之力”产生异能的人，可以有效自卫以及保护“品器”不落入他人之手。


“阅之国”乃东之大陆冉冉升起的新兴国家，其中有个“哇牛叉”学院，就是“阅之国”专门设立的专门设计、生产与使用“品器”的机构，“哇牛叉”学院又分“沪江总院”和“京城分院”，每个学院又分两个社团，前端公社和设计公社。

而本故事讲述的就是“哇牛叉”学院中异能的伙伴们之间的故事。

	<p>精灵泉之力平时不外放，目前已知下面两种场景会外放：感受到威胁逃跑的时候会在外部包裹一层泉之力，推测是自我防御机制，也有说法可以让逃跑速度加快；另外一种就是想办法引起精灵的情感共振，因为精灵本身就是灵魂体，带有生前的情感，因此，如果引发共鸣会让“泉之力”如泪水般不自主涌出，“情感共振”越强，“泉之力”释放量就会越多。</p>

	<p>“精神共振”可以人，环境产生，但效果最好利用率最好的还是使用“品器”来“精神共振”，</p>
	<p>精灵按大类分为“白精灵”和“黑精灵”，以及非常罕见的“蓝精灵”。比例关系如下：“白精灵”和“黑精灵”比例大约为100比1，根据地域不同，差不多10万到100万个精灵中才会出现一个“蓝精灵”。</p>
	<p>1. “白精灵”的纹理是浅灰色，纹路总共5~6种类型，纹路密度各不相同；</p>
	<p>2. “黑精灵”体色暗淡，纹路为纯黑色的，应该是饱含冤屈的灵魂体形成， “黑精灵”也可以释放“泉之力”，但“泉之力”包含怨气，非常不纯，如果被常人吸收堪比中毒，会导致异能障碍，有时候后果非常严重，因此非常不受欢迎。但某些黑暗组织据说专门用这种包含怨气“泉之力”提升自己的异能。“黑精灵”可以直接从外部吸收“泉之力”，这个和“白精灵”不一样，“白精灵”只能内部产生内部吸收；</p>
	<p>3. “蓝精灵”的纹理是蓝色，据传，普通白精灵吸收了至少100单位“泉钻”才能让浅灰色纹理质变成蓝色，似乎发生了升级。也正因为如此，具有大量的“灵力”，一次释放可以抵得上上万普通“白精灵”。但是，根据计算，就算是自然灵气最丰富的地方，一个普通灵魂体要想变成“蓝精灵”大约需要百年的时间，所谓“百年轮回”或许就是这个意思。但是，确实有人见到“蓝精灵”，推测，或许是某位异能大能死后的灵魂体所成，因为异能大能自身就包含大量“泉之力”。更有这么一种说法，精灵一旦成为“蓝精灵”，就会主动寻找灵魂脱壳的活死人，自己就会占据这个肉体，重新复活。如果真是这样，也可以理解为什么精灵总是一刻不停拼命吸收“自然灵气”寻求成长了。</p>
	<p>精灵散布在东之大陆的各个地方，可谓无处不在，但是密度差异明显，其中有三处最为富饶，分别是北方的“京城”，东侧的“沪江”，南部的“深广”，由于“泉之力”只能由精灵释放，因此，这三处富饶之地乃大国必争之处。</p>

	<dt><strong>泉之力</strong></dt>
	<dd>一种凝练“泉钻”必不可少的灵力，名叫“泉”，人们总喜欢合称为“泉之力”。纯净的“泉之力”只能由白精灵释放。
	<p>国家的强盛与否与“泉之力”的多少并无直接关系，关键要看是否要强大的把“泉之力”凝练成“泉钻”晶体的能力。有一些国家，虽然“泉之力”不算很多，但凝练能力异常强大，其实力也是相当强悍，如一些名为“游之国”的国家；还有的国家通过透支“泉钻”吸收了很多“泉之力”，但说不定只是强弩之末，例如以前闻名一时的“团之国”。</p>
	</dd>
	<dt><strong>泉钻</strong></dt>
	<dd>互联世界最重要的物质，即使货币，又是一种能量来源。其存储量往往决定了一个国家的强盛与否。“泉钻”暴露在空气中，其能量会慢慢释放到大气中，直至消失不见。因此，“泉钻”需要放在专门用做存储的“品器”中，这类“品器”以坚固和稳定著称。“泉钻”只能由名为“泉”的灵力（人们称之为“泉之力”）凝练而成。“泉钻”计量单位为“克拉”，1单位“泉钻”晶体仅有小拇指甲头那么大，但是，别小看这么一点“泉钻”，其代表的财力已经足够锻造一个1阶甚至2阶的品器了。
	<p>对于普通人而言，“泉钻”是财富；对于异能者而言，则是快速提升能力的宝物，同口鼻吸收“泉之力”效率低下，但是，直接吞噬“泉钻”可以快速释放“泉之力”被身体吸收，然后再运功凝结成“泉钻”，存储在身体中。体内“泉钻”越多，功力越强，异能威力也更大更持久。</p>
	</dd>
	<dt><strong>品器</strong></dt>
	<dd>由能人异士锻造的专门处理“泉之力”或“泉钻晶体”的器物，颜色形状材质以及作用原理各异，按照品阶分为1-9阶，9品最强，只能吸收很少部分精灵的“泉之力”。按照功能，往往分为2大类：一是“攻坚型品器”，常年在外对战、采集和凝结“泉之力”使用；二是“存储型品器”，功能比较单一，就是存储凝练成的“泉钻”，异常兼顾与稳定，并有重兵把守。</dd>
	<dd>由于很多东西可以让精灵产生“精神共振”，因此，攻坚型“品器”种类成千上万，但基本上工作方式都是一样的，通过各种手段吸引精灵注意，然后用简单便捷的方式对精灵进行“精神共振”，迅速吸收其释放的“泉之力”。</dd>
</dl>
